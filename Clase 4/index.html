TRABAJANDO CON SVG

Rosario Morales & Antonia Tapia

Las imágenes SVG son vectores creados o leidos como código de fuente. Se utiliza un lenguaje similar al HTML para indicar colores, posiciones, grosores de linea y demás. Hace las imágenes facilmente editables y que conserven calidad visual.
